Headers

Expected values

Accept-Language

Should be a single language code (e.g., "en", "es", etc.), followed by q= and a value between 0 and 1.

When fingerprinting protections are disabled, the listed language codes should match each language set in brave://settings/languages.

In default fingerprinting settings, this value should match only the first language set in brave://settings/languages.

In aggressive fingerprinting settings, this value should always be en, regardless of the language settings in brave://settings/languages.
